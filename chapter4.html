Chapter-4:Programming in C

Working with function

a. Define Function: A function is a block of code that performs a specific task and can be reused throughout the program.

b. Syntax of Function:

return_type function_name(parameter_list) {
    // code block
}

c. Types of Functions:

  • Library Functions: Predefined functions like printf(), scanf(), etc.
  • User-defined Functions: Created by the programmer to perform specific tasks.

d. Components of Function:

  • i. Function Prototype: Declaration of function before main.
    int sum(int, int);
  • ii. Function Call: Calling the function inside main or another function.
    sum(2, 3);
  • iii. Function Definition: Actual implementation of the function.
int sum(int a, int b) {
    return a + b;
}

iv. Return Type: The data type the function returns (e.g., int, float, void).


2. Categories of Functions with Example:

i. Function with return type but no arguments:

int getValue() {
    return 10;
}

ii. Function with return type with arguments:

int sum(int a, int b) {
    return a + b;
}

iii. Function with no return type with no arguments:

void greet() {
    printf("Hello!");
}

iv. Function with no return type with arguments:

void printSquare(int n) {
    printf("%d", n * n);
}

3. Storage Classes:

i. Automatic (auto): Default for local variables, automatically created and destroyed.

void func() {
    auto int x = 5;
}

ii. External (extern): Declares a global variable defined elsewhere.

extern int x;

iii. Register: Suggests storing variable in CPU register for speed.

register int i;

iv. Static: Retains value between function calls.

void count() {
    static int x = 0;
    x++;
    printf("%d", x);
}

4. Recursive Function (with syntax and example):

#include 
int factorial(int n) {
    if (n == 0)
        return 1;
    else
        return n * factorial(n - 1);
}

int main() {
    int num = 5;
    printf("Factorial of %d is %d\n", num, factorial(num));
    return 0;
}

5. Demonstration of Passing Array to a Function:

#include <stdio.h>

void display(int arr[], int size) {
    for(int i = 0; i < size; i++)
        printf("%d ", arr[i]);
}

int main() {
    int nums[] = {1, 2, 3, 4};
    display(nums, 4);
    return 0;
}

Structure in C

Introduction and Syntax

A structure is a user-defined data type in C that allows grouping variables of different types under one name.

struct Student {
    int id;
    char name[50];
    float marks;
};

Structure Size

The size of a structure is the sum of sizes of its members plus any padding added for alignment.

Accessing Members of Structure

struct Student s1;
s1.id = 1;
strcpy(s1.name, "John");
s1.marks = 85.5;

Nested Structure

struct Date {
    int day, month, year;
};

struct Student {
    int id;
    struct Date dob;
};

Array of Structures

struct Student students[3];
students[0].id = 1;

Passing Structure to Function

void display(struct Student s) {
    printf("%d %s %.2f", s.id, s.name, s.marks);
}

Union in C

Introduction and Syntax

Union allows storing different data types in the same memory location, but only one member can contain a value at any given time.

union Data {
    int i;
    float f;
    char str[20];
};

Comparison between Structure and Union

Structure: Memory allocated for all members.
Union: Shared memory for all members (size of largest member).

Pointers in C

Introduction and Syntax

int *ptr;

Concept of Value and Address

int a = 10;
int *p = &a;
printf("%d", *p); // value
printf("%p", p);  // address

Declaration and Initialization

int a = 5;
int *p = &a;

Pointer and Function

void print(int *p) {
    printf("%d", *p);
}

Call by Reference

void update(int *x) {
    *x = *x + 1;
}

Call by Value vs Call by Reference

Value: Function gets a copy, original not changed.
Reference: Function modifies original value.

Pointer with Arrays

int arr[5] = {1,2,3,4,5};
int *p = arr;

Advantages and Disadvantages

+ Efficient memory handling
- Can lead to segmentation faults if misused

File Handling in C

Concept of Data File

Files store data permanently unlike variables which store in memory temporarily.

Need for File Handling

To read/write data even after the program ends.

Types of Files

Sequential and Random Access Files

File Handling Functions

FILE *fopen(const char *filename, const char *mode);
int fclose(FILE *stream);
int getc(FILE *stream);
int putc(int char, FILE *stream);
int fprintf(FILE *stream, const char *format, ...);
int fscanf(FILE *stream, const char *format, ...);
int getw(FILE *stream);
int putw(int w, FILE *stream);
char *fgets(char *str, int n, FILE *stream);
int fputs(const char *str, FILE *stream);
size_t fread(void *ptr, size_t size, size_t count, FILE *stream);
size_t fwrite(const void *ptr, size_t size, size_t count, FILE *stream);
int remove(const char *filename);
int rename(const char *oldname, const char *newname);

Random Access Functions

int fseek(FILE *stream, long offset, int origin);
void rewind(FILE *stream);
long ftell(FILE *stream);

File Opening Modes

"r"    - Read
"r+"   - Read/Write
"w"    - Write (overwrite)
"w+"   - Read/Write (overwrite)
"a"    - Append
"a+"   - Read/Append

Steps to Work with File

1. Declare FILE pointer: FILE *fp;
2. Open file: fp = fopen("file.txt", "w");
3. Read/Write using functions
4. Close file: fclose(fp);

Reading from File

char ch;
while ((ch = fgetc(fp)) != EOF) {
    putchar(ch);
}

Writing to File

fprintf(fp, "Hello World");

Appending Data to File

fp = fopen("file.txt", "a");
fputs("New Line", fp);

End of File (EOF)

EOF is a marker to indicate end of file.
